## üéÆ Game Modes Details

- **Difficulty**: 3x3 (Easy), 4x4 (Medium), 5x5 (Hard)

- **Objective**: Place all pieces in correct positions

- **Difficulty**: 8, 12, or 18 pairs
- **Scoring**: Based on matches and time

- **Objective**: Find all hidden words

- **Difficulty**: 8x8, 12x12, or 16x16 grids
